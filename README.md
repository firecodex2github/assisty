#  Ticket Management System

A Ticket Management System built using **JavaScript**, **HTML**, **CSS**, and **Bootstrap**.  
This application supports user authentication, ticket creation with file uploads, and secure token-based session handling.


##  Features

###  Authentication
- User Login & Registration
- JWT Access Token + Refresh Token Flow
- Secure Session Storage Handling
- Auto Token Refresh

###  Ticket Management
- Create Support Tickets
- Upload Screenshots & PDF Files
- Role-Based Access

###  Frontend
- Responsive UI using Bootstrap
- Floating Label Forms
- Modal Notification System
- File Preview Before Upload
- Client-Side Validation



### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
